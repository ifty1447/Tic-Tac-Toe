# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using Python.

## 🚀 Features
- Two-player gameplay
- Interactive command-line interface
- Win/draw detection system

## 🛠️ Technologies Used
- Python

## 📂 Project Structure
- TicTacToeGame.py → Main game file
- TicTacToe-png.png → Game preview image

## ▶️ How to Run
1. Install Python
2. Download the project
3. Run the file:
   python TicTacToeGame.py
